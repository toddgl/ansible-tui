# ansible-tui
An Ansible Tui build using py-textual

## Requirements

* > py-textual
* > py-yaml

## Key Bindings

| Key       | Action                |
| ----------| ----------------------|
| Tab       | Switch panels         |
| Space     | Toggle host / role    |
| a         | Select all hosts      |
| v         | Entre Vault password  |
| c         | Toggle --check        |
| d         | Toggle --diff         |
| r         | Run playbook          |
| q         | Quit                  |
